# グランナチュール（Grande NATURE）LP

## 1. プロジェクト概要
- 東京都世田谷区奥沢にあるエステサロン「グランナチュール」の1ページランディングページ
- ターゲット：自由が丘・奥沢エリアでリラクゼーションを求める20-50代女性
- 目的：初回限定メニューの訴求とWEB予約への導線強化

## 2. ページ構成
- `/` - 1ページLP（ファーストビュー、About、人気メニュー、基本情報、アクセス、予約CTA、フッター）

## 3. コア機能
- [x] ファーストビュー（ヒーロー）
- [x] About / コンセプト紹介
- [x] 人気メニュー5品の表示
- [x] 基本情報（営業時間・定休日・TEL・最寄駅）
- [x] アクセス情報（地図・住所）
- [x] 予約CTAボタン
- [x] SNS・外部リンク（Instagram、LINE、公式サイト、予約サイト）
- [x] スクロールアニメーション

## 4. データモデル
- データベース不要（静的LP）

## 5. バックエンド/外部連携
- Supabase: 不要
- Shopify: 不要
- Stripe: 不要
- 外部リンク: Instagram, LINE公式, 公式サイト, 予約サイト(Reservia)
